15 r""" 

16 Simplify combinatorial expressions. 

17 

18 This function takes as input an expression containing factorials, 

19 binomials, Pochhammer symbol and other "combinatorial" functions, 

20 and tries to minimize the number of those functions and reduce 

21 the size of their arguments. 

22 

23 The algorithm works by rewriting all combinatorial functions as 

24 gamma functions and applying gammasimp() except simplification 

25 steps that may make an integer argument non-integer. See docstring 

26 of gammasimp for more information. 

27 

28 Then it rewrites expression in terms of factorials and binomials by 

29 rewriting gammas as factorials and converting (a+b)!/a!b! into 

30 binomials. 

31 

32 If expression has gamma functions or combinatorial functions 

33 with non-integer argument, it is automatically passed to gammasimp. 

34 

35 Examples 

36 ======== 

37 

38 >>> from sympy.simplify import combsimp 

39 >>> from sympy import factorial, binomial, symbols 

40 >>> n, k = symbols('n k', integer = True) 

41 

42 >>> combsimp(factorial(n)/factorial(n - 3)) 

43 n*(n - 2)*(n - 1) 

44 >>> combsimp(binomial(n+1, k+1)/binomial(n, k)) 

45 (n + 1)/(k + 1)
