10class beta(Function): 

11 r""" 

12 The beta integral is called the Eulerian integral of the first kind by 

13 Legendre: 

14 

15 .. math:: 

16 \mathrm{B}(x,y) := \int^{1}_{0} t^{x-1} (1-t)^{y-1} \mathrm{d}t. 

17 

18 Beta function or Euler's first integral is closely associated with gamma function. 

19 The Beta function often used in probability theory and mathematical statistics. 

20 It satisfies properties like: 

21 

22 .. math:: 

23 \mathrm{B}(a,1) = \frac{1}{a} \\ 

24 \mathrm{B}(a,b) = \mathrm{B}(b,a) \\ 

25 \mathrm{B}(a,b) = \frac{\Gamma(a) \Gamma(b)}{\Gamma(a+b)} 

26 

27 Therefore for integral values of a and b: 

28 

29 .. math:: 

30 \mathrm{B} = \frac{(a-1)! (b-1)!}{(a+b-1)!} 

31 

32 Examples 

33 ======== 

34 

35 >>> from sympy import I, pi 

36 >>> from sympy.abc import x,y 

37 

38 The Beta function obeys the mirror symmetry: 

39 

40 >>> from sympy import beta 

41 >>> from sympy import conjugate 

42 >>> conjugate(beta(x,y)) 

43 beta(conjugate(x), conjugate(y)) 

44 

45 Differentiation with respect to both x and y is supported: 

46 

47 >>> from sympy import beta 

48 >>> from sympy import diff 

49 >>> diff(beta(x,y), x) 

50 (polygamma(0, x) - polygamma(0, x + y))*beta(x, y) 

51 

52 >>> from sympy import beta 

53 >>> from sympy import diff 

54 >>> diff(beta(x,y), y) 

55 (polygamma(0, y) - polygamma(0, x + y))*beta(x, y) 

56 

57 We can numerically evaluate the gamma function to arbitrary precision 

58 on the whole complex plane: 

59 

60 >>> from sympy import beta 

61 >>> beta(pi,pi).evalf(40) 

62 0.02671848900111377452242355235388489324562 

63 

64 >>> beta(1+I,1+I).evalf(20) 

65 -0.2112723729365330143 - 0.7655283165378005676*I
