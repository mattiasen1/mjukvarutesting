32 Variable Intervals 

33 ================== 

34 

35 The basic format is [var, min, max, steps], but the

57 Coordinate Modes 

58 ================ 

59 

60 Plot supports several curvilinear coordinate modes, and 

61 they independent for each plotted function. You can specify 

62 a coordinate mode explicitly with the 'mode' named argument, 

63 but it can be automatically determined for Cartesian or 

64 parametric plots, and therefore must only be specified for 

65 polar, cylindrical, and spherical modes. 

66 

67 Specifically, Plot(function arguments) and Plot[n] = 

68 (function arguments) will interpret your arguments as a 

69 Cartesian plot if you provide one function and a parametric 

70 plot if you provide two or three functions. Similarly, the 

71 arguments will be interpreted as a curve if one variable is 

72 used, and a surface if two are used. 

73 

74 Supported mode names by number of variables: 

75 

76 1: parametric, cartesian, polar 

77 2: parametric, cartesian, cylindrical = polar, spherical 

78 

79 >>> Plot(1, mode='spherical')
